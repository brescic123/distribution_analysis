Distribution Analysis – TV Price Data
Opis projekta:
Ovaj projekat analizira distribuciju cijena televizora i prodajnu efikasnost proizvoda koristeći Python biblioteke Pandas i NumPy. Cilj analize je razumjeti varijabilnost cijena, stabilnost brendova, prodajne trendove i odnos između ocjena proizvoda i njihove prodaje.
Korištene biblioteke:
pandas
numpy
Analiza uključuje:
učitavanje i pripremu dataseta
izračun raspona cijena (range) između najskupljeg i najjeftinijeg televizora
izračun interkvartilnog opsega (IQR)
identifikaciju najprodavanijeg televizora i njegove pozicije u distribuciji
grupisanje proizvoda po brendovima
sortiranje proizvoda po standardnoj devijaciji cijena
identifikaciju 5 brendova sa najujednačenijim cijenama
analizu prosječnog broja prodanih jedinica po kvartilima ocjena proizvoda
Cilj analize:
Prepoznati brendove sa stabilnom cjenovnom politikom
Razumjeti distribuciju prodaje i ocjena proizvoda
Pružiti uvid u efikasnost proizvoda i njihovu popularnost
Napomena:Projekat je dio mog data analytics portfolija i demonstrira sposobnost analize poslovnih podataka kroz Python.
